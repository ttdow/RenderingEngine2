\doxysubsection{Detailed Description}
A simple 3D vector class for representing and manipulating 3D vectors. 

Supports common vector operations such as addition, scaling, normalization, dot and cross products, and interpolation. Uses a union for both named access (\textquotesingle{}x\textquotesingle{}, \textquotesingle{}y\textquotesingle{}, \textquotesingle{}z\textquotesingle{}) and array-\/style access via \textquotesingle{}data\mbox{[}3\mbox{]}\textquotesingle{}.
